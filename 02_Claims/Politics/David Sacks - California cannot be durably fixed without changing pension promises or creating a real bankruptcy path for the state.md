# David Sacks - California cannot be durably fixed without changing pension promises or creating a real bankruptcy path for the state

## Position
David Sacks argues that California cannot be durably fixed without changing pension promises or creating a real bankruptcy path for the state.

## Evidence Cited
- He says state-court precedent blocks California from cutting pension benefits promised to public employees.
- He says those promises create an enormous long-term fiscal overhang for the state.
- He says the only real escape valves are a constitutional change to the pension structure or some path to state bankruptcy.

## Logic Chain
1. Sacks treats pension obligations as legally locked costs that normal budget politics cannot unwind.
2. If those liabilities cannot be modified, only extraordinary institutional change can reset the state balance sheet.
3. That supports the claim that California needs pension reform or a bankruptcy path to be durably fixed.
