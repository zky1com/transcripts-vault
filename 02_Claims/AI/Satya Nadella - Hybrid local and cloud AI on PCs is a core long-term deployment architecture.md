# 
Satya Nadella - Hybrid local and cloud AI on PCs is a core long-term deployment architecture

## Position
Microsoft is pursuing a hybrid architecture where local PC models handle parts of workload and cloud models handle larger tasks.

## Evidence Cited
- Nadella says Microsoft is committed to making PCs strong for local models.
- He discusses workstation-class local compute relevance returning.
- He describes local prompt processing plus cloud calls as an active direction.

## Logic Chain
1. Local inference improves responsiveness and data locality for some tasks.
2. Cloud inference provides elastic scale for heavier workloads.
3. Hence hybrid local-cloud execution is a pragmatic long-term architecture.
