# 
Satya Nadella - Multi-model orchestration is becoming the default pattern for enterprise AI applications

## Position
Enterprise AI apps are moving to orchestrating multiple models per workflow instead of relying on a single foundation model.

## Evidence Cited
- Nadella says builders will use all models, not one model.
- He gives an orchestration example where role-assigned models improve outcomes.
- He compares model evolution to database market specialization.

## Logic Chain
1. Different models have different strengths and tradeoffs.
2. Workflow-level routing can outperform one-model pipelines.
3. Therefore multi-model orchestration is a likely default architecture.
