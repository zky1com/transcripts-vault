# 
Satya Nadella - Microsoft is extending Microsoft 365 identity systems to support agent workers

## Position
Microsoft is productizing agent identity in Microsoft 365 so organizations can operate non-human workers under enterprise control frameworks.

## Evidence Cited
- Nadella references introducing agent identity capability in Microsoft 365.
- He says this extends existing human identity and endpoint protection systems.
- He describes both delegated and independent agent identity modes.

## Logic Chain
1. Organizations already govern human users through identity platforms.
2. Agent workers need the same governance plane to run in production.
3. So extending M365 identity to agents is a direct enterprise adoption path.
