# 
Satya Nadella - Coding workflows now combine chat, actions, and autonomous agents

## Position
AI coding is now a composed workflow where suggestions, chat, actions, and autonomous agents are used together rather than as a single interface.

## Evidence Cited
- Transcript describes progression from next-edit suggestions to chat, then actions, then autonomous agents.
- Nadella says developers use foreground and background agents while still editing directly in VS Code/CLI.
- He frames modern coding as combining form factors in parallel.

## Logic Chain
1. Coding subtasks have different control and latency needs.
2. Different AI modes handle different subtasks better.
3. So practical coding productivity comes from orchestration across modes.
