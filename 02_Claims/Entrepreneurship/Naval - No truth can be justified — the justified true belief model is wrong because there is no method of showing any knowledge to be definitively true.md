# Naval - No truth can be justified — the justified true belief model is wrong because there is no method of showing any knowledge to be definitively true

## Position

The dominant model of knowledge — "justified true belief" — says that you know something when you have justified it as true. This is still the most prevalent view, shared by Bayesians who think they know something when they have accumulated enough confirming evidence to be very confident. The problem is fundamental: there is no method of showing any piece of knowledge to be definitively true. Every theory, no matter how well-supported, remains susceptible to refutation by as-yet-undiscovered evidence. Euclid's geometry appeared definitively true for 2,000 years; Newtonian physics appeared definitively true for 300 years. Both were eventually shown to have limits. The very concept of "justified true belief" creates a false ceiling on knowledge: once you've "proved" something true, you treat it as unfalsifiable, which is a religious stance disguised as epistemology. The correct view is that all knowledge is conjectural — best-guess that has survived serious attempts at refutation, not certain truth.

- "if it's true that means there is nothing false about it and therefore it can't possibly be refuted it's a very religious notion"

## Logic Chain

1. Justified true belief: know X = have justified X as true
2. But there is no method of showing any X to be definitively true — history shows all "proved" theories eventually found limited or wrong
3. Treating X as "proven true" → treating it as unfalsifiable → religious stance
4. This blocks future improvement: if X is true, we cannot be looking for ways X is wrong
5. Correct model: all knowledge is conjectural — surviving best-guess that has resisted refutation so far
6. This keeps all knowledge open to improvement, making progress possible without limit
