# Naval - Code and media are permissionless leverage with no marginal cost of replication

## Position
Code and media (podcasts, writing, video) are the most important modern forms of leverage because they have zero marginal cost of replication. Unlike labor (needs followers) or capital (needs investors), code and media require no one's permission. Anyone can record a podcast, write a book, or build software and reach millions without asking for approval. This makes them egalitarian: everyone has access to the same leverage tools.
