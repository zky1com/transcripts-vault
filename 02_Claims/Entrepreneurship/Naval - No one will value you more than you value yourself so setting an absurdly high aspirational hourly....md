# Naval - No one will value you more than you value yourself so setting an absurdly high aspirational hourly...

## Position

Wealth creation requires setting a personal hourly rate that feels absurdly high and committing to it even before you have money. Naval set his at $5,000/hour before having real money, which retrospectively proved to be ~$1,000/hour in reality. If the rate doesn't feel absurd it's not high enough. The self-valuation precedes and shapes the external valuation.

## Evidence Cited

- Personal anecdote: Naval set $5k/hour aspirational rate for himself before having money
- Retrospective: the actual earned rate proved to be ~$1,000/hour
- Anecdote of making theatrical shows of throwing things away or donating rather than returning them
