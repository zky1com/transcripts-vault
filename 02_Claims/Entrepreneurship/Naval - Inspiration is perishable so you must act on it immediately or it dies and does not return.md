# Naval - Inspiration is perishable so you must act on it immediately or it dies and does not return

## Position

Inspiration has a very short shelf life. If you hesitate when inspired to write, build, or create, the motivation evaporates and you won't return to it. Naval has unpublished tweet storms that died this way. The correct strategy is to act immediately when inspired, not to schedule or defer the action.

## Evidence Cited

- Personal anecdote: Naval has unpublished tweet storms that died because he paused
- Corollary: impatience with actions, patience with results

## Logic Chain

1. Inspiration is a transient emotional/cognitive state, not a renewable resource on command

4. Therefore the window to act productively is at peak inspiration
