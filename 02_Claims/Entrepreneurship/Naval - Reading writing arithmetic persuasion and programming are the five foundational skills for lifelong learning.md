# Naval - Reading writing arithmetic persuasion and programming are the five foundational skills for lifelong learning

## Position

If forced to name the minimum set of skills that set a person up to learn and operate effectively for life, Naval identifies five: reading, writing, arithmetic, persuasion (speaking), and programming. The first three are classical foundations that have always been the minimum for functional literacy. Persuasion is added because the ability to communicate ideas verbally is a distinct skill from writing and equally essential in a world that runs on human coordination. Programming is added because it is applied arithmetic — it gives the same logical and mathematical thinking as pure math but in a form that generates immediate, high-leverage outputs in almost any domain. The claim is not that these are sufficient for success in any specific domain, but that possessing all five creates a platform from which any domain becomes learnable.

## Evidence Cited

- "the five most important skills are of course reading, writing, arithmetic — and then as you are adding in persuasion which is talking — and then finally I would add computer programming just because it's an applied form of arithmetic that just gets you so much leverage for free in any domain"
- "if you're good with computers, if you're good at basic mathematics, if you're good at writing, if you're good at speaking, and if you like reading, you're set for life"
- "practical persuasion and a deep understanding of some complex topic — I think you'll have a great foundation for learning for the rest of your life"

## Logic Chain

1. Reading: absorbs knowledge from the entire accumulated record of human thought
2. Writing: forces clarity, enables knowledge creation and communication
3. Arithmetic (mathematics): provides logical foundation for evaluating truth claims in any domain
4. Persuasion: enables coordination with other humans — essential in any collaborative endeavor
5. Programming: applied arithmetic — translates mathematical thinking into high-leverage outputs; increasingly applicable in every domain
6. The set is not arbitrary: each skill is foundational rather than domain-specific → they compound across all future learning
7. "Set for life" = not needing a fixed career path — can learn and adapt into any new domain
