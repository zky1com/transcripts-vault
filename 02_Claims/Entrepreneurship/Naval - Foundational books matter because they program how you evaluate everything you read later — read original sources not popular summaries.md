# Naval - Foundational books matter because they program how you evaluate everything you read later — read original sources not popular summaries

## Position

There is a class of books that deserves special priority: original foundational texts in their fields. Naval recommends Adam Smith over business books, Darwin's Origin of Species over modern evolution books, Feynman's Six Easy Pieces over popular cosmology. The reason is structural: these books program the framework through which you will evaluate all subsequent reading in that domain. If you read the derivative summaries first, your evaluative framework is built on second-hand interpretations. If you read the original, you understand the actual argument from the source, and you can therefore evaluate later derivatives correctly. The analogy Naval uses is building on a steel frame: if you have genuine foundational understanding, you will not be afraid of any book in the library. If you only have memorized derivatives, you are lost as soon as the situation departs from what you memorized.

## Evidence Cited

- "there are actually things you can read especially early on that will program your brain a certain way and then later things that you read you will decide whether those things were true or false based on the earlier things"
- "instead of reading a business book pick up Adam Smith's The Wealth of Nations instead of reading a book on biology or evolution that's written today I would pick up Darwin's Origin of Species"
- "you should be able to walk into a library and not fear any book"

## Logic Chain

1. Early books program the evaluative framework for a domain — they determine what you think is true or false later
2. If your framework is built on popular summaries: you inherit someone else's interpretation, which may distort the original
3. If your framework is built on original foundational texts: you have the primary argument, can evaluate everything else against it
4. Steel frame analogy: foundation must be derived from first principles, not memorized — otherwise you cannot adapt when situations change
5. Goal: walk into any library and fear no book — possible only with genuine foundational understanding, not derivative memorization
