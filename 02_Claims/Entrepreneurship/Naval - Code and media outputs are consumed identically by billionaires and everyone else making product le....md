# Naval - Code and media outputs are consumed identically by billionaires and everyone else making product le...

## Position

Jeff Bezos cannot get better Google search results, better Netflix content, or a meaningfully better smartphone experience than an ordinary person. Code and media products are structurally identical across all consumer segments. This makes product leverage unique: the same output serves billionaires and middle-class consumers equally. In contrast, labor-intensive goods (vacations, food preparation, massages) scale with wealth.
