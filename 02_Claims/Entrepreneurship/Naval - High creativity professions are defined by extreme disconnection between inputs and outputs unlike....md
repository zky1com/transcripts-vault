# Naval - High creativity professions are defined by extreme disconnection between inputs and outputs unlike...

## Position

In physical labor (e.g., lumberjacking without tools), the best worker is maybe 3x more productive than the worst. In creative/leveraged professions (e.g., software engineering), one engineer can create Bitcoin and billions of dollars of value while another can work a full year and produce code nobody wants. The input-output disconnect is unlimited in creative work and capped in physical work.

## Evidence Cited

- Software engineer creating Bitcoin vs engineer shipping unused code for a year
- Lumberjack example: best vs worst capped at ~3x without tools
- Tools and leverage (bulldozers, software) unlock the disconnection

## Logic Chain

1. Physical work productivity variance is constrained by human biomechanics
2. Creative/leveraged work productivity variance is unconstrained â€” ideas scale infinitely
3. The distinction is leverage: tools, code, media amplify the disconnect
4. Therefore, to maximize wealth potential, enter a profession where input-output disconnect is maximum
5. This is structurally the same as "get into a leveraged position"
