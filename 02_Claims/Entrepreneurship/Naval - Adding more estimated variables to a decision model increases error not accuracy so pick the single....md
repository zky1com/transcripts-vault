# Naval - Adding more estimated variables to a decision model increases error not accuracy so pick the single...

## Position

When each variable in a model is an estimate, multiplying more estimates together produces apparent precision (decimal points) that is entirely spurious. This is "false precision" â€” the error compounds with each added variable rather than averaging out. The practical consequence for career and wealth decisions is that you should identify the one or two variables with the highest signal (e.g., "what am I genuinely good at that the market values") and ignore the rest. Simpler models with fewer estimated inputs are more reliable than complex ones.

## Logic Chain

1. Each estimated input variable carries irreducible uncertainty
2. Multiplying uncertain estimates propagates and amplifies uncertainty
3. More variables = more compounded uncertainty, regardless of how precise the arithmetic appears
4. Therefore: fewer, higher-quality inputs produce more reliable outputs than many low-quality inputs
5. Applied to career decisions: one strongly confirmed variable (natural skill + market value) predicts success better than a 10-factor model
