# Naval - Falsifiability is necessary but not sufficient for a good explanation — any theory can be made to f...

## Position

Deutsch's key insight (relayed by Naval): falsifiable theories are "a dime a dozen" â€” they tell you nothing about the quality of the explanation. The classic example is "eat 1.0 kilograms of grass to cure the common cold." This is testable, but no one should test it because there is no mechanistic explanation for *why* grass would cure a cold. And when it fails, the theory can absorb the failure by adjusting a parameter: "try 1.1 kilograms," "try a different kind of grass," "try a different day." The theory remains technically falsifiable but is never actually falsified because its parameters are infinitely adjustable. A genuinely good explanation requires more than testability â€” it must be hard to vary.

## Logic Chain

1. Popper established falsifiability as a demarcation criterion for science â€” a theory must be testable to count as scientific
2. But testability alone is insufficient: the grass-cure-for-cold theory is testable but has no mechanistic explanation
3. Any failed test can be explained away by adjusting a free parameter (dose, timing, variety) â€” the theory has infinite adjustment room
4. This means falsifiability without a hard-to-vary structure produces no epistemic progress
5. Therefore: good explanations require both testability *and* a mechanistic structure that constrains the free parameters
