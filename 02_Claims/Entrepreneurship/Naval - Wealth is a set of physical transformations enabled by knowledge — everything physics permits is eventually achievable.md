# Naval - Wealth is a set of physical transformations enabled by knowledge — everything physics permits is eventually achievable

## Position

Deutsch defines material wealth as the set of physical transformations that knowledge makes possible — not as a fixed stock of goods or resources, but as the space of possible actions on the physical world. This definition ties wealth directly to knowledge: more knowledge = larger space of possible transformations = more wealth. The corollary is that the upper bound on achievable wealth is set by the laws of physics, not by the current stock of resources. Everything that is not forbidden by physics is eventually achievable through knowledge creation. This is the strongest possible anti-Malthusian position: scarcity is not a physical constraint but a knowledge constraint, and knowledge constraints can always be relaxed by creating more knowledge.

## Evidence Cited

- "material wealth is a set of physical transformations that we can affect"
- "everything that is not forbidden by the laws of physics is eventually possible through knowledge and knowledge creation"
- "the universe is ours as a resource to learn about and exploit"

## Logic Chain

1. Material wealth = set of physical transformations we can currently perform
2. What we can transform depends on what we know how to do
3. More knowledge → more transformations possible → more wealth
4. The hard ceiling is the laws of physics: what they forbid is truly impossible
5. Everything else — even what currently seems impossible — is potentially achievable with sufficient knowledge
6. Therefore: wealth grows without limit as knowledge grows without limit
7. This makes the resource-scarcity argument a knowledge-scarcity argument — and knowledge creation has no ceiling
