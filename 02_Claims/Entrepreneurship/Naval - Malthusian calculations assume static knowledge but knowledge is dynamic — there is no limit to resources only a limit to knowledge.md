# Naval - Malthusian calculations assume static knowledge but knowledge is dynamic — there is no limit to resources only a limit to knowledge

## Position

Malthusian-style resource calculations share a common methodological flaw: they treat the current technology stack as permanent and project its consumption of known resources into the future. This produces apparently airtight proofs that civilization will collapse — food, wood, coal, oil will run out. But these calculations assume the knowledge needed to use alternative resources and to reduce consumption of scarce ones will not be created. That assumption is reliably wrong. Every time a "running out" problem has seemed acute, knowledge has advanced in ways that dissolved the problem — not by finding more of the same resource, but by making the resource irrelevant, by finding substitutes, or by using it far more efficiently. The real limit is not matter but knowledge. Pessimism about human creativity is not a neutral empirical stance; it is an unfounded assumption that the future will resemble the present in the one variable (knowledge) that has never remained static.

## Evidence Cited

## Logic Chain

1. Malthusian model: resource R is finite, consumption rate C > R/T → R runs out in T/C years
2. Model is valid given: current technology is permanent and no substitutes will be found
3. Both assumptions fail historically — technology changes and new substitutes are always found
4. The hidden variable is knowledge: what we know determines what counts as a resource
5. As knowledge grows, previously useless materials become resources; previously scarce resources become replaceable
6. Therefore: the binding constraint is not matter (which is effectively infinite) but knowledge (which is expandable without limit)
