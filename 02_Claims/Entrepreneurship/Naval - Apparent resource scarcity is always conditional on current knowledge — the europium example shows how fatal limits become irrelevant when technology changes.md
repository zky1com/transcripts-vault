# Naval - Apparent resource scarcity is always conditional on current knowledge — the europium example shows how fatal limits become irrelevant when technology changes

## Position

Resource scarcity is never an absolute fact about nature; it is always relative to the current technology stack. The europium example from Deutsch's Beginning of Infinity illustrates this precisely. Sixty years ago, europium was the only element that could produce the red phosphor needed for color cathode ray tube televisions. Mathematically robust calculations showed it would run out. The calculations were technically correct within their assumptions. The assumption that turned out to be wrong was that cathode ray tubes would continue to be the relevant technology. When LCD screens replaced them, the entire europium scarcity problem became irrelevant. No europium needed. This pattern recurs throughout technological history: an apparently absolute resource constraint that seemed to impose a hard limit on civilization dissolved when the underlying technology changed. The constraint was real given current knowledge; it was not real given the knowledge that would be created.

## Logic Chain

1. Technology A (cathode ray tubes) requires resource X (europium) with no substitutes
2. X is finite → mathematical proof that X will run out → technology A will be impossible
3. Logic is valid given assumption: technology A is permanent
4. Assumption fails: technology B (LCD) doesn't use X → resource constraint dissolves
5. The "shortage" was not a fact about nature but a fact about current knowledge
6. This pattern applies to all resource constraints: they dissolve when knowledge advances
