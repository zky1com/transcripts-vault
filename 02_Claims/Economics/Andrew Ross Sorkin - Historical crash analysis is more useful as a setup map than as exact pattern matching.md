# Andrew Ross Sorkin - Historical crash analysis is more useful as a setup map than as exact pattern matching

## Position
The practical lesson from 1929 is understanding setup conditions, not expecting a replay with identical sequence and timing.

## Evidence Cited
- Transcript asks whether today is another 1929 and repeatedly emphasizes context comparison.
- Discussion reviews preconditions like rates, leverage, and policy backdrop rather than exact chart analogs.
- Conversation focuses on mechanisms and incentives that transfer across eras.

## Logic Chain
1. Market structure changes across eras.
2. But incentive and leverage dynamics recur.
3. So historical use is strongest for framework building, not one-to-one prediction.
