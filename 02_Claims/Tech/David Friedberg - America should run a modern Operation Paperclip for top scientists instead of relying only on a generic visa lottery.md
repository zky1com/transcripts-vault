# David Friedberg - America should run a modern Operation Paperclip for top scientists instead of relying only on a generic visa lottery

## Position
David Friedberg argues that America should run a modern Operation Paperclip for top scientists instead of relying only on a generic visa lottery.

## Evidence Cited
- He says the US previously recruited about 1,600 German scientists and engineers after World War II.
- He says China now publishes the majority of papers across many scientific domains.
- He says the US should proactively identify top industries and scientists and recruit them with government and private-industry support.

## Logic Chain
1. Friedberg distinguishes strategic scientific recruitment from broad labor-market visa processing.
2. He argues that frontier talent competition requires active targeting, not just passive visa channels.
3. That supports the claim that America should run a modern Operation Paperclip.
