# Long-horizon stock-level return concentration evidence supports constituent-first analysis over broad top-down index simplification

## Fact
- U.S. equity wealth creation over long horizons is highly concentrated in a minority of companies, so index outcomes are strongly shaped by constituent-level fundamentals.

## Source Excerpt
- Bessembinder's study shows the majority of net wealth creation above T-bills came from a relatively small subset of stocks.
