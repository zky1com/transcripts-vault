# Dimensional documentation describes DFUS as actively managed while Vanguard documentation describes VTI as index-tracking

## Core Evidence
Dimensional materials describe DFUS as an actively managed ETF implementation, while Vanguard materials describe VTI as an ETF that seeks to track a broad U.S. equity index.
