# Equal-weight index construction and concentration metrics show cap-weight headline index moves can diverge from broader participation

## Fact
- S&P equal-weight indexes assign the same weight to each constituent, while cap-weight indexes concentrate exposure in the largest firms, so relative performance between them is a valid participation/breadth diagnostic.

## Source Excerpt
- S&P methodology states equal-weight indices use the same constituents as parent indices but equal constituent weights, contrasting with cap-weighted construction.
